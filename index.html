Installing Erlang

Fastes way to start using Erlang is to install the precompiled packages either from erlang.org or Erlang Solutions.

Setting PATH environment variable

After Erlang have been installed you need to add it to system environment.

Windows 8.1

  1. press windows button + s and search for system environment
  2. press Edit the system variable
  3. press Environment Variables button
  4. Search in system variables list for Path variable. Mark it and press edit. Add your path to Erlang into the long list.

Windows 7

  1. press windows button + s and search for system environment
  2. press Edit the system variable
  3. press Environment Variables button
  4. Search in system variables list for Path variable. Mark it and press edit. Add your path to Erlang into the long list.

Editors

Emacs

With Erlang install there is a erlang-mode that can be added to Emacs.

Erl IDE

There is a Erl IDE that should work to Eclipse or Intellij.

Tools on Windows

Here is some tools for developing Erlang on Windows.

Git

This vsn is used much of Erlang projects. Can be installed from here.
After installing it add it to path in the same way as the erlang install.

Rebar3

Rebar3 is a good tool to create projects, compile and make releases.
To read how to use and how to install it follow the instructions on their homepage rebar3
